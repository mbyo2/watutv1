## 🚀 **Quick Start**

### **📋 Prerequisites**
- **Node.js 16+**
- **npm** or **yarn**
- **Modern web browser**

### **🔧 Installation**
```bash
# Clone the repository
git clone https://github.com/mbyo2/watutv1.git
cd watutv1

# Install dependencies
npm install

# Start development server
npm run dev

# Open in browser
# http://localhost:5173
```

### **🏗️ Production Build**
```bash
# Build for production
npm run build

# Serve with any web server
# Files are in dist/ folder
```
